# dataservice
ejs service that provides data via rest apis

# setup on server
1. create folder dataservice
2. go into the folder
3. init
npm init
4. install express
npm install express -g
5. create server.js
6. run server
node server.js
7. install csvtojson
npm install csvtojson --save
8. install cors
npm install cors
